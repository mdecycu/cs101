cs1graphics.pdf

cs1graphics is a Python module specifically designed for introductory computer science courses, particularly those focusing on object-oriented programming and graphics. It provides a simple and intuitive way to create and manipulate graphical objects on a canvas, making it easier for beginners to visualize and understand programming concepts.

Key features and concepts of cs1graphics include:

Canvas:

The Canvas class represents the drawing window where graphical objects are displayed.

Drawable Objects:

cs1graphics offers various classes for creating different shapes and elements, such as Circle, Rectangle,Text, Image, Polygon, and Path. These objects can be manipulated by setting properties like size, color, position, and border styles.

Object-Oriented Principles:

The library is built upon object-oriented principles, allowing students to learn about classes, objects, methods, and attributes in a practical context.

Animation:

It supports basic animation by allowing objects to be moved and transformed over time, often by repeatedly redrawing the canvas after changing object properties.

Accessibility:

It aims to simplify graphics programming for beginners, abstracting away the complexities of underlying graphics libraries like Tkinter or PIL. 

In essence, cs1graphics serves as a pedagogical tool to introduce fundamental programming concepts through visual and interactive graphics creation.

cs1media.pdf

cs1media is a Python library primarily designed for educational purposes, specifically for introductory computer science courses (often referred to as CS1). It provides a simplified interface for image manipulation, allowing students to easily load, modify, and display images.

Key features of cs1media:

Image Loading and Display:

It enables users to load images from files and display them in a window.

Pixel-level Manipulation:

Users can access individual pixels within an image and modify their RGB color values. This allows for various image processing techniques like grayscale conversion, color replacement, or applying filters.

Simplified Interface:

Compared to more complex image processing libraries, cs1media offers a straightforward set of functions and methods, making it accessible for beginners.

Educational Context:

It is commonly used in environments like Pythonpad, which is an embeddable Python programming environment designed for web-based CS1 exercises.

from cs1media import *

# Load an image
img = load_picture("example.png")

# Get image dimensions
width, height = img.size()

# Convert image to grayscale
for y in range(height):
    for x in range(width):
        r, g, b = img.get(x, y)
        gray = int(0.299 * r + 0.587 * g + 0.114 * b)
        img.set(x, y, (gray, gray, gray))

# Display the modified image
img.show()

cs1robots.pdf

cs1robots is a Python library designed for teaching introductory programming concepts using a robot simulation. It provides a simple, grid-based world where a virtual robot can perform basic actions like moving, turning, picking up, and putting down beepers. This library is inspired by RUR-PLE and aims to make learning programming more engaging through visual interaction.

Key features and usage:

Robot actions:

The cs1robots library allows users to program a robot to perform fundamental actions such as move(), turn_left(), pick_beeper(), and put_beeper().

World interaction:

Robots can interact with walls and beepers within a customizable world.

Visual feedback:

The library provides a graphical representation of the robot's actions and the world, making it easy to visualize program execution.

Educational focus:

It is primarily used in educational settings to introduce concepts like loops, conditionals, and functions in a practical and visually intuitive manner.

Install cs1robots:

pip install cs1-robots

from cs1robots import *

# Create a world (or load an existing one)
create_world() 

# Create a robot instance
hubo = Robot()

# Program the robot's actions
hubo.move()
hubo.turn_left()
hubo.pick_beeper()
